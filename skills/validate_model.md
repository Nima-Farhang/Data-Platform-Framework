# Skill: Validate Model

## Purpose
Review a dbt model for structural quality and standards alignment.

## Checks
- naming clarity
- correct layer placement
- config reasonableness
- use of shared macros
- incremental logic safety
- audit metadata usage
- test recommendations

## Output
1. summary
2. strengths
3. issues
4. recommendations

## Quality Rules
- feedback must be concrete
- feedback should improve maintainability
- feedback should align with platform principles
